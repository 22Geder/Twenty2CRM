## 🔧 API Endpoints חדשים

### POST `/api/candidates/auto-tag`
זיהוי אוטומטי של תגיות מקורות חיים

**Body:**
```json
{
  "candidateId": "uuid",
  "resumeText": "קורות חיים בטקסט..."
}
```

**Response:**
```json
{
  "success": true,
  "matchedTags": 5,
  "tags": ["React", "Node.js", "Python", ...]
}
```

### GET `/api/candidates/auto-tag?tags=React,Python&minExperience=3&minRating=4`
סינון מועמדים לפי קריטריונים
