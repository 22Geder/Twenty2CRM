# 📱 Twenty2Jobs - אפליקציית מובייל PWA

## מה נוצר?

אפליקציית PWA (Progressive Web App) מלאה שעובדת על **Galaxy**, **iPhone** ומחשב.

---

## 🚀 איך להתקין על הטלפון

### Galaxy / Android:
1. פתחו את הקישור בדפדפן Chrome: `https://your-domain.com/upload-cv`
2. יופיע באנר "התקן עכשיו" - לחצו עליו
3. **או**: הקישו על שלוש הנקודות ⋮ → "הוסף למסך הבית" / "Install app"
4. האפליקציה תותקן ותופיע במסך הבית!

### iPhone / iPad:
1. פתחו את הקישור בדפדפן **Safari** (חשוב!)
2. לחצו על כפתור השיתוף ⬆️ (למטה באמצע)
3. גללו ובחרו **"הוסף למסך הבית"**
4. לחצו "הוסף" בפינה העליונה
5. האפליקציה תופיע במסך הבית!

---

## 📤 איך לשתף קו"ח מ-WhatsApp

### Galaxy / Android:
1. פתחו קובץ קו"ח ב-WhatsApp
2. לחצו על שלוש הנקודות ⋮ או כפתור השיתוף
3. בחרו **"Twenty2"** או **"Twenty2Jobs"** מרשימת האפליקציות
4. הקובץ יועלה אוטומטית והמועמד יתווסף למערכת!
5. תקבלו התראה על הצלחה

### iPhone:
1. פתחו קובץ קו"ח ב-WhatsApp
2. לחצו על כפתור השיתוף ⬆️
3. בחרו **"Twenty2"** מהאפליקציות
4. הקובץ יועלה אוטומטית!

> 💡 **טיפ**: ניתן לשתף קבצי PDF, Word (doc/docx) וקבצי טקסט

---

## 🔔 התראות

האפליקציה תשלח לכם התראות כשתשתפו קבצים בהצלחה.
לאישור התראות, היכנסו לאפליקציה ואשרו כשתתבקשו.

    └── create-icon-placeholders.js   # יצירת אייקונים

---

## ⚙️ הגדרות Share Target

ב-`manifest.json`:
```json
"share_target": {
  "action": "/api/share-target",
  "method": "POST",
  "enctype": "multipart/form-data",
  "params": {
    "files": [{
      "name": "file",
      "accept": [".pdf", ".doc", ".docx", ".txt"]
    }]
  }
}
```

## 🧪 בדיקה מקומית

```bash
cd crm-app
npm run dev
```

- מהטלפון: `http://[IP-המחשב]:3000/upload-cv`
- לרשת מקומית: `npm run dev:network`

## 🔧 פתרון בעיות

### האפליקציה לא מופיעה בשיתוף:
1. ודאו שהתקנתם מ-Chrome (אנדרואיד) או Safari (iPhone)
2. נסו להסיר ולהתקין מחדש
3. רעננו את הדף לפני ההתקנה

### השיתוף לא עובד:
1. ודאו שיש חיבור אינטרנט
2. בדקו שהקובץ הוא PDF/Word/TXT
3. נסו להעלות ישירות מהאפליקציה

### אין התראות:
1. בדקו הרשאות בהגדרות הטלפון
2. היכנסו לאפליקציה ואשרו התראות
