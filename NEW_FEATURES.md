## 1️⃣ פייפליין Kanban חזותי 🎯

**איפה?** דאשבורד → 🎯 פייפליין Kanban

### מה זה עושה?
- **גרירה חופשית** - גרור מועמדים בין שלבים
- **6 שלבים:** חדש → סינון → ראיון → הצעה → התקבל / נדחה
- **מידע מלא** - כל מועמד מציג: שם, טלפון, מייל, ציון התאמה, תגיות
- **עדכון אוטומטי** - כל שינוי נשמר מיד ונרשם בהיסטוריה

### איך להשתמש?
1. פתח: `/dashboard/kanban`
2. גרור כרטיס מועמד לשלב החדש
3. השרת מעדכן אוטומטית! ✅

### שלבים:
- 📥 **חדש** - קורות חיים חדשים
- 🔍 **סינון** - עבר סינון ראשוני
- 💼 **ראיון** - בתהליך ראיונות
- 📝 **הצעה** - נשלחה הצעת עבודה
- ✅ **התקבל** - התקבל לעבודה!
- ❌ **נדחה** - לא התאים

## 2️⃣ היסטוריית פעילות 📋

**איפה?** דאשבורד → 📋 היסטוריית פעילות

### מה זה עושה?
- **רישום אוטומטי** של כל פעולה במערכת
- **מעקב אחר מי עשה מה ומתי**
- **Timeline ויזואלי** - רואים את ההיסטוריה בצורה ברורה

### מה נרשם?
- ✅ שינוי שלב מועמד
- 📝 הוספת הערות
- 📧 שליחת מיילים/SMS
- 📅 תזמון ראיונות
- 📎 העלאת מסמכים

### איך להשתמש?
1. פתח: `/dashboard/activity`
2. רואה את כל הפעילות - **חדש ← ישן**
3. לחץ על שמות לעבור למועמדים/משרות

## 3️⃣ תבניות הודעות 📝

**איפה?** דאשבורד → 📝 תבניות הודעות

### מה זה עושה?
- **תבניות מוכנות** להודעות SMS, מיילים, ווואטסאפ
- **משתנים אוטומטיים** - `{name}`, `{positionTitle}`, `{companyName}`
- **ניהול קל** - יצירה, עריכה, מחיקה

### איך להשתמש?
1. פתח: `/dashboard/templates`
2. לחץ **+ תבנית חדשה**
3. מלא:
   - שם התבנית (לדוגמה: "הזמנה לראיון")
   - סוג (SMS / מייל / וואטסאפ)
   - תוכן עם משתנים: `שלום {name}, אנו מזמינים אותך לראיון למשרת {positionTitle}...`
4. שמור ✅

### דוגמאות לתבניות:

**דחייה מנומסת:**
```
שלום {name},

תודה שהגשת מועמדות למשרת {positionTitle} אצלנו.
לאחר בחינה מעמיקה, החלטנו להמשיך עם מועמדים אחרים.

בהצלחה בהמשך,
{companyName}
```

**הזמנה לראיון:**
```
שלום {name}! 🎉

מצאנו את הפרופיל שלך מעניין למשרת {positionTitle}.
נשמח לקבוע אתך ראיון ב-{date} בשעה {time}.

אנא אשר הגעה בהקדם.

צוות {companyName}
```

## 4️⃣ תזכורות אוטומטיות ⏰

**איפה?** דאשבורד → ⏰ תזכורות

### מה זה עושה?
- **תזכורות למעקב** - follow-ups, שיחות, ראיונות
- **ניהול מוקדם** - תאריך יעד + עדיפות
- **סימון כהושלם** - checkbox פשוט

### איך להשתמש?
1. פתח: `/dashboard/reminders`
2. לחץ **+ תזכורת חדשה**
3. מלא:
   - כותרת: "להתקשר ליוסי"
   - תאריך: מתי לבצע
   - עדיפות: נמוכה / בינונית / גבוהה / **דחוף**
   - תיאור: פרטים נוספים
4. המערכת תציג לך תזכורות לפי דחיפות! 🔔

### סוגי תזכורות:
- 📞 **מעקב** - follow-up עם מועמד
- 💼 **ראיון** - ראיון מתוזמן
- ☎️ **שיחה** - להתקשר
- 📧 **מייל** - לשלוח מייל
- 📌 **כללי** - כל דבר אחר

## 5️⃣ ניקוד מועמדים 🌟

**איפה?** בעמוד מועמד ספציפי

### מה זה עושה?
- **ציון 0-100** לכל מועמד
- **צבעים ויזואליים** - אדום (חלש) → ירוק (מצוין)
- **מיון לפי ציון** - מיין מועמדים לפי איכות

### איך להשתמש?
1. פתח מועמד: `/dashboard/candidates/[id]`
2. תראה **ציון מועמד** בראש הדף
3. לחץ ✏️ **ערוך**
4. גרור הסליידר או הקלד ציון (0-100)
5. לחץ 💾 **שמור**

### סולם ציונים:
- 🟢 **80-100** - מצוין (excellent!)
- 🔵 **60-79** - טוב
- 🟡 **40-59** - בינוני
- 🟠 **20-39** - חלש
- 🔴 **0-19** - לא מתאים

---

## 6️⃣ הערות פנימיות 📌

**איפה?** בעמוד מועמד ספציפי

### מה זה עושה?
- **הערות פרטיות** על מועמדים
- **4 סוגי הערות** - כללי, משוב ראיון, שיחת טלפון, בדיקת ממליצים
- **הערות פרטיות** 🔒 - רק אתה רואה

### איך להשתמש?
1. פתח מועמד: `/dashboard/candidates/[id]`
2. גלול ל**📌 הערות פנימיות**
3. לחץ **+ הערה חדשה**
4. כתוב את ההערה
5. בחר סוג + סמן "פרטי" אם צריך
6. שמור ✅

### דוגמאות:
- **כללי:** "מועמד מעולה, רמת אנגלית גבוהה"
- **משוב ראיון:** "עבר ראיון טכני - 9/10, תקשורת מצוינת"
- **שיחת טלפון:** "דיברנו ב-15/12, רוצה 25K, זמין מיידי"
- **בדיקת ממליצים:** "מנהל קודם אישר - עובד מצוין"
