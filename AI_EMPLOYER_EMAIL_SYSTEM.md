## 📊 10 פרמטרי ניתוח חכמים

המערכת משתמשת ב-**10 אלגוריתמים** שונים ליצירת משפטי ההתאמה:

### 1. 🏷️ התאמת תגיות בסיסית
```javascript
if (תגיות תואמות > 70%) {
  "מביא/ה ניסיון מוכח ב-React, Node.js, TypeScript..."
}
```

### 2. 🔍 התאמה חלקית של תגיות
```javascript
if ("JavaScript" מתאים ל-"JS") {
  כולל התאמה במשפט
}
```

### 3. 💼 ניסיון רלוונטי
```javascript
if (ניסיון >= 5 שנים) {
  "רקע עשיר ויכולת להשתלב במהירות"
} else if (ניסיון >= 2) {
  "ניסיון מעשי מאפשר לתרום מיד"
}
```

### 4. 🎯 תפקיד נוכחי
```javascript
if (תפקיד_נוכחי דומה למשרה) {
  "כיום Senior Developer ב-[חברה], הניסיון מתאים במדויק"
}
```

### 5. 📍 מיקום גיאוגרפי
```javascript
if (אותה עיר) {
  "מיקום אידיאלי - ממוקם/ת ב-[עיר], קרוב למשרדי החברה"
} else if (אותו אזור) {
  "מיקום נוח באזור [אזור]"
}
```

### 6. 💰 התאמת שכר
```javascript
if (ציפיות שכר תואמות) {
  "ציפיות השכר מתאימות לתקציב ומאפשרות התקדמות חלקה"
}
```

### 7. ⏰ זמינות
```javascript
if (זמין מיידית) {
  "זמינות מיידית - יכול/ה להתחיל בהקדם"
} else {
  "תקופת הודעה של [X], מאפשרת תכנון מסודר"
}
```

### 8. 📋 איכות קורות חיים
```javascript
if (קורות חיים מלאים) {
  "פרופיל מקצועי ומלא, עם קורות חיים מסודרים"
}
```

### 9. ⭐ דירוג
```javascript
if (דירוג >= 4) {
  "דירוג גבוה במערכת ([X]/5) מעיד על איכות יוצאת דופן"
}
```

### 10. 🎓 כישורים נוספים
```javascript
if (מיומנויות נוספות רלוונטיות) {
  "מיומנויות כמו [X] ו-[Y] מוסיפות ערך משמעותי"
}
```
